# Детекция и классификация животных на видео

Проект по компьютерному зрению для обнаружения и классификации животных на видео.  
Алгоритм выделяет движущиеся объекты на кадрах, отслеживает их между кадрами и классифицирует как курицу или овцу с помощью обученной нейросети EfficientNet.

## Описание проекта

Входное видео обрабатывается покадрово. Сначала выделяются движущиеся объекты, затем они отслеживаются между кадрами и классифицируются с помощью обученной модели.

Основные этапы:

1. Детекция движущихся объектов (background subtraction)
2. Отслеживание объектов между кадрами
3. Классификация объектов нейросетью EfficientNet
4. Визуализация результатов на видео

На выходе формируется видео с выделенными животными и подписью класса.

## Алгоритм

### 1. Детекция движения

Для поиска объектов используется алгоритм обнаружения движения:

- строится фон методом скользящего среднего
- вычисляется разность текущего кадра и фона
- применяется пороговая обработка
- используются морфологические операции

### 2. Отслеживание объектов

Для сопоставления объектов между кадрами используется:

- IoU (Intersection over Union)
- алгоритм Венгера (Hungarian algorithm)

### 3. Классификация

Каждый найденный объект периодически классифицируется нейросетью EfficientNet-B0.

Классы:

- 0 - background
- 1 - chicken
- 2 - sheep

Низкоуверенные предсказания отбрасываются.

### 4. Визуализация

Для каждого обнаруженного животного:

- рисуется ограничивающий прямоугольник
- выводится:
  - ID объекта
  - класс
  - вероятность

## Установка

Клонировать репозиторий:

```bash
git clone https://github.com/7renko/animal-video-detection.git
cd animal-video-detection
```
Установить зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

Запуск обработки видео:

```bash
python detection.py
```

Скрипт:

1. Считывает видео из
```markdown
data/animals.mp4
```

2. Выполняет детекцию и классификацию объектов

3. Сохраняет результат в
```markdown
results/result_animals.mp4
```

## Модель

Используется нейросеть EfficientNet-B0, обученная на датасете из трёх классов:

- background

- chicken

- sheep

Конфигурация обучения находится в:
```markdown
model/configs/animals.yaml
```
